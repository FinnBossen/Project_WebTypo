Same-origin policy im Browser

  • Schränkt Requests an andere Domains als aktuelle Domain ein
  • Sicherheitsmassnahme aktueller Browser
  • Bezug generative Gestaltung: Verhindert das Laden von Ressourcen von anderen Domains innerhalb von Scripten (Processing, Three.js, Paper.js) bei der Nutzung von z.b. Open Data.
  • Umgehen via CORS oder Umleitung und Weiterleitung der Request über eigenen Server (innerhalb des geladenen Domains)

Audio im Browser

  • Moderne Browser verhindern das Abspielen von Audio ohne Userinput
  • Canvas Elemente müssen eine "Geste" (z.b. Klick) emfangen, bevor sie Audio abspielen dürfen
  • Implementation der Einschränkung Browserabhängig: Komplettes Verbot Audiowiedergabe zu starten bis Geste (Chrome, Opera, Safari) oder Stummschaltung (aber Wiedergabe) bis zur Aktivierung von Audio (Firefox, Edge, IE). → Audiowiedergabe im Canvas erst nach emfangen einer Geste starten, um das Starten der Wiedergabe zu garantieren.
